DIAGRAMAS DE CLASE
EN DIA UML

ING. RENÉ ELIZALDE

Dia es una aplicación informática de propósito general para la creación de diagramas, desarrollada como parte del proyecto GNOME creada originalmente por Alexander Larsson. Para utilizarlo no se requiere ni un ordenador potente ni grandes conocimientos técnicos.


Se puede utilizar para dibujar diferentes tipos de diagramas. Actualmente se incluyen diagramas como: diagramas UML, diagramas de flujo, diagramas de redes, diagramas de circuitos eléctricos, etc.

En principio Dia fue diseñado para sistemas operativos Linux. Sin embargo, en la actualidad el programa también ofrece una versión para Windows que podemos descargar de Internet.

Entre las características de Dia podemos mencionar:

    Permite crear una gran variedad de mapas y diagramas: diagramas de flujo, circuitos eléctricos, puzles, redes, cronogramas, etc.
    Es un programa ligero que no requiere grandes prestaciones. Funciona ágilmente en la mayoría de ordenadores.
    Ofrece mucha libertad al usuario para personalizar los trabajos.
    Permite exportar los archivos a distintos formatos.
